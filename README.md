# web-based data wrangling and visualization using Streamlit and a variety of libraries

An interactive web app that enables users to clean, transform, and visualize their data.

## link to the app

[click here to see the app](#) → replace this line with your actual Streamlit link after deploying it on an online server.

## running the code on your own machine

1. Pull down a copy of this repo.
2. Download required packages:
   Pip install -r requirements.txt
3. Run the code:
   Streamlit run app.py

## available functions in this application

There is streamlit based application which is used for upload of data, cleaning and transforming, and finally a third page is available to Build dynamic charts. There are six different types of charts you can Build in Page c including:

Histogram
Box plot
Scatter chart
Line graph
Bar chart
Correlation matrix

and last one is used for downloading the project.
